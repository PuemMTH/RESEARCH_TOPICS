# วิทยานิพนธ์

## อาจารย์ที่ปรึกษา

ผู้ช่วยศาสตราจารย์ ดร. หัชทัย ชาญเลขา

## หัวข้อวิทยานิพนธ์

การสร้างคำอธิบายรูปภาพและกราฟในเอกสารวิชาการโดยใช้บริบทจากเนื้อหาเอกสาร

## รายละเอียดโดยสังเขป

งานวิจัยนี้มีแนวคิดในการพัฒนากระบวนการสร้างคำอธิบายรูปภาพและกราฟในเอกสารวิชาการ โดยใช้ข้อมูลจากทั้งรูปภาพและบริบทของเนื้อหาในเอกสารร่วมกัน เนื่องจากในปัจจุบันระบบสรุปเอกสารหรือแบบจำลองภาษาขนาดใหญ่มักให้ความสำคัญกับข้อความเป็นหลัก ในขณะที่องค์ความรู้บางส่วนในเอกสารวิชาการอยู่ในรูปภาพ กราฟ ตาราง หรือแผนภาพ ซึ่งหากไม่มีการอธิบายอย่างเหมาะสม อาจทำให้ข้อมูลสำคัญบางส่วนสูญหายไป

งานวิจัยนี้จึงมุ่งเน้นการวิเคราะห์องค์ประกอบของเอกสาร เช่น รูปภาพ กราฟ คำบรรยายภาพ และข้อความรอบข้าง เพื่อนำมาสร้างคำอธิบายที่มีความหมายเชิงวิชาการมากกว่าการบรรยายภาพทั่วไป โดยใช้เทคโนโลยีด้าน Computer Vision, Natural Language Processing และ Multimodal Learning ผลลัพธ์ที่คาดหวังคือกระบวนการที่สามารถช่วยสร้างคำอธิบายรูปภาพหรือกราฟใน paper ได้อย่างสอดคล้องกับบริบทของงานวิจัย และสามารถนำไปใช้ต่อยอดในระบบสรุปเอกสาร การจัดการองค์ความรู้ และการช่วยอ่านเอกสารวิชาการได้

## วัตถุประสงค์

1. เพื่อศึกษาแนวทางการสร้างคำอธิบายรูปภาพและกราฟในเอกสารวิชาการ
2. เพื่อพัฒนากระบวนการวิเคราะห์รูปภาพร่วมกับบริบทของเนื้อหาในเอกสาร
3. เพื่อสร้างคำอธิบายรูปภาพหรือกราฟที่สามารถสะท้อนสาระสำคัญเชิงวิชาการของเอกสารได้
4. เพื่อประเมินคุณภาพของคำอธิบายที่สร้างขึ้นในด้านความถูกต้อง ความสอดคล้องกับบริบทของเอกสาร และความสามารถในการถ่ายทอดสาระสำคัญจากรูปภาพหรือกราฟ

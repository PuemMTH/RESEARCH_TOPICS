Leveraging Author-Specific Context for Scientific Figure Caption Generation: 3rd SciCap Challenge

งานนี้เป็นตัวอย่างล่าสุดที่เอา LaMP-Cap ไปใช้จริงใน challenge โดยผสม context filtering, prompt optimization และ profile-informed refinement เพื่อแก้ปัญหา caption ที่ยาว หลุดบริบท หรือไม่ตรง writing style

  • zero-shot prompting ด้วย paragraph information สร้าง caption ที่มี irrelevant content และ hallucination
  • scientific caption ต้อง balance ระหว่าง factual accuracy, conciseness และ author-specific style
  • paper หลาย domain ต้องใช้ prompt ที่ปรับตาม category ไม่ใช่ template เดียวสำหรับทุก field

Key Contributions

  • Two-stage pipeline: Stage 1 ทำ content-grounded caption generation, Stage 2 ทำ profile-informed stylistic refinement
  • Prompt optimization: ใช้ DSPy MIPROv2 และ SIMBA สร้าง category-focused prompts
  • Candidate selection: ใช้ Gemini-2.5-flash เป็น reranker เพื่อเลือก caption ที่ precision และ style ดีขึ้น
How It Works

Methodology

1

Stage 1: Content-Grounded Generation

เริ่มจาก sentence-level filtering เพื่อลด noise ใน paragraph จากนั้นใช้ category-focused prompt optimization สำหรับ paper category ต่าง ๆ เช่น CS, Math, Physics, Statistics และใช้ candidate selection เมื่อ paper มีหลาย category

2

Stage 2: Profile-Informed Refinement

นำ caption จาก Stage 1 ไปปรับด้วย few-shot prompting จาก profile figures โดย enforce ความยาวประมาณ ±15% เพื่อให้ caption กระชับ และใช้ profile captions เป็น structural/style reference

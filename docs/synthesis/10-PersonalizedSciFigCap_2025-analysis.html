Key Contributions

  • Quality evaluator: fine-tune Qwen-2.5-VL-3B ให้ประเมิน caption quality จาก score 1-6 โดยเทียบกับ GPT-4.1 labels
  • Personalized generator: fine-tune Qwen-2.5-VL-7B ด้วย target figure + paragraph + mention + OCR + profile data
  • Quality-aware generation: ทดลองให้โมเดล predict quality score ก่อนสร้าง caption เพื่อควบคุม trade-off ระหว่าง style กับ quality

Methodology

1

Quality Filtering

ผู้เขียนใช้ GPT-4.1 สร้าง synthetic quality assessment labels จำนวน 3,000 ตัวอย่าง จากนั้น fine-tune Qwen-2.5-VL-3B เป็น quality evaluator ได้ Spearman correlation 0.759 และ QWK 0.754 บน validation set

2

Personalized Caption Generator

generator คือ Qwen-2.5-VL-7B รับ target figure, explanatory paragraph, textual mentions, OCR tokens และ profile data จาก paper เดียวกัน แล้ว fine-tune บน LaMP-Cap โดยใช้เฉพาะ sample ที่ quality score >= 3

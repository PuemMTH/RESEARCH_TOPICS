LaMP-Cap: Personalized Figure Caption Generation With Multimodal Figure Profiles

Paper นี้สำคัญกับ thesis เพราะเปลี่ยนโจทย์ figure captioning จากการดูเฉพาะ target figure ไปสู่การใช้ profile figures จาก paper เดียวกันเป็นบริบท multimodal เพื่อสร้าง caption ที่ใกล้ author-written caption มากขึ้น

Key Contributions

  • LaMP-Cap dataset: สร้างจาก SciCap Challenge โดยให้ target figure พร้อม multimodal source และ profile figures สูงสุด 3 รูปจาก paper เดียวกัน
  • Context profile formulation: profile แต่ละชุดมี image, caption และ figure-mentioning paragraph เพื่อแทนทั้งเนื้อหาและ style
  • Empirical finding: profile information ช่วยให้ caption ใกล้ ground-truth มากขึ้น และ profile image มีประโยชน์มากกว่า paragraph ใน ablation

Methodology

Dataset Construction

LaMP-Cap curate จาก SciCap Challenge Dataset ซึ่งมี figures จาก arXiv papers จำนวนมาก แล้วเลือกหนึ่งรูปเป็น target figure และใช้รูปอื่นใน paper เดียวกันเป็น profile context โดยตัด paper ที่มีรูปเดียวออก เพราะไม่สามารถสร้าง target-profile pair ได้

  • 110,828 target figures
  • 307,903 total figures รวม target และ profile
  • profile สูงสุด 3 รูปต่อ target figure
  • split เป็น train/validation/test แบบ 80/10/10
2

Input Format

โมเดลได้รับ target image และ figure-mentioning paragraph เป็น source จากนั้นเพิ่ม profile figures จาก paper เดียวกันเพื่อช่วยเรื่อง context, domain vocabulary และ author style การทดลองเปรียบเทียบ No-Profile, One-Profile และ All-Profile

Gaps & Future Work

ข้อจำกัดและทิศทางต่อไป

  • dataset จำกัด profile ไม่เกิน 3 รูป จึงยังไม่ครอบคลุม long-document context เต็ม paper
